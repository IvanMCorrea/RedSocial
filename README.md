## Deploy

<!-- DEMO: [Netlify Deploy][deploy-url] -->

#### 1. Clone the repo

```sh
git clone https://github.com/IvanMCorrea/E-commerce
```

#### 2. Install NPM packages

```sh
npm install
```

#### 3. Enter API-KEYS (.env)

#### 4. Deploy

```sh
npm start
```
